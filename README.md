# showing-warning-lucid
Repo to show a "annoying" warning in Lucid

## First
Create one .env for a database mysql (I am using 5.7.33)

## Second 
Recreate database - node ace migration:fresh

## Third 
Run node ace db:seed 

Already: Warning 👀
